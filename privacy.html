개인정보처리방침

SOS Bridge는 해외여행 중 갑작스러운 아픔이나 부상 상황에서 도움 요청을 돕는 참고 안내 서비스입니다. SOS Bridge는 의료기관, 응급기관 또는 공공기관이 아닙니다.

SOS Bridge는 의료진의 진단이나 치료를 대체하지 않습니다. 응급 상황에서는 즉시 현지 응급번호로 연락하거나 가까운 의료기관, 현지 공공기관, 숙소, 대사관 또는 영사관의 도움을 받으세요.
SOS Bridge는 의료기기가 아니며, 질병이나 건강 상태를 진단, 치료, 완화 또는 예방하기 위한 서비스가 아닙니다.

1. 서비스 소개

SOS Bridge는 한국어로 상황을 정리하고, 선택한 여행 국가/지역의 언어로 도움 요청 문장과 여행자 의료카드를 보여주는 보조 도구입니다. 사용자가 현지 의료진, 주변 사람, 숙소 직원, 공공기관에 상황을 설명할 수 있도록 돕는 것이 목적입니다.

2. 처리하는 정보

현재 앱 구조 기준으로 사용자가 직접 입력하거나 브라우저 기능을 통해 처리될 수 있는 정보는 다음과 같습니다.

  • 여행 국가/지역 선택 정보
  • 여행 도시/지역 입력 정보
  • 증상 입력 내용
  • 의료카드 입력 정보: 이름 또는 별칭, 여권상 영문 이름, 국적, 나이, 혈액형, 알레르기, 복용 중인 약, 기존 질환, 비상 연락처, 여행자 보험, 숙소 주소
  • 사용자가 위치 권한을 허용하는 경우의 현재 위치 정보
  • 브라우저 또는 기기에서 자동으로 처리될 수 있는 기본 기술 정보

3. 정보를 사용하는 목적

  • 여행 국가/지역 설정과 현지어 도움 문장 표시
  • 여행자 의료카드 작성, 표시, 현지어 번역
  • 응급번호 정보와 정식 지원/제한 지원 상태 표시
  • 증상 입력을 바탕으로 한 AI Care 참고 안내 제공
  • 병원·약국 검색 기준 제공 및 지도 또는 길찾기 연결
  • 서비스 안정성 확인

4. 저장 여부

현재 베타 버전에서는 별도 회원가입 계정 서버나 데이터베이스에 의료카드 정보와 증상 정보를 저장하지 않습니다.

현재 베타 버전에서는 여행자 의료카드 정보와 의료카드 번역 캐시가 사용자의 브라우저 localStorage에 저장될 수 있습니다. 이 정보는 SOS Bridge의 별도 회원 계정 서버에 저장되지 않으며, 사용자는 앱 안의 “의료카드 정보 삭제하기” 버튼 또는 브라우저 사이트 데이터 삭제 기능으로 삭제할 수 있습니다.

현재 앱에서 sessionStorage 사용은 확인되지 않았습니다. 현재 앱 자체에서 쿠키를 저장하는 별도 로직도 확인되지 않았습니다.

5. 개인정보의 처리 및 보유 기간

  • 브라우저 localStorage에 저장되는 의료카드 정보와 번역 캐시는 사용자가 삭제 버튼을 누르거나 브라우저 사이트 데이터를 삭제할 때까지 남아 있을 수 있습니다.
  • 별도 회원 계정 서버 또는 데이터베이스에 저장하지 않는 정보는 서버 보유 기간이 적용되지 않습니다.
  • 외부 API를 통해 처리되는 정보는 해당 외부 서비스의 처리 정책과 기술적 처리 과정에 따라 일시적으로 처리될 수 있습니다.
  • SOS Bridge가 향후 계정 기능, 서버 저장 기능, 유료 기능을 추가하는 경우 개인정보처리방침을 업데이트합니다.

6. 개인정보의 파기 절차 및 방법

  • 사용자는 앱 안의 “의료카드 정보 삭제하기” 버튼으로 브라우저에 저장된 의료카드 정보를 삭제할 수 있습니다.
  • 브라우저 설정에서 사이트 데이터 또는 localStorage 데이터를 삭제할 수 있습니다.
  • 현재 베타 버전에서 별도 서버 데이터베이스에 저장하지 않는 정보에는 별도의 서버 파기 절차가 적용되지 않습니다.
  • 향후 서버 저장 기능이 추가되는 경우 별도 파기 절차를 고지합니다.

7. 외부 서비스 및 API 사용

현재 코드에서 확인되는 외부 처리 또는 연결은 다음과 같습니다.

  • Netlify Functions: AI Care와 번역 요청을 중계하기 위해 사용됩니다.
  • OpenAI API: 사용자가 입력한 증상 또는 번역 대상 문장을 처리하기 위해 사용될 수 있습니다.
  • Overpass API 및 Nominatim: 병원·약국 검색 또는 위치 기반 검색에 사용될 수 있습니다.
  • Google Maps 링크: 길찾기 화면 연결에 사용될 수 있습니다.
  • 브라우저 위치 권한: 사용자가 허용한 경우에만 현재 위치를 검색 기준으로 사용합니다.

8. 개인정보 처리업무의 외부 처리 가능성

SOS Bridge는 현재 별도 회원 계정 데이터베이스를 운영하지 않지만, 번역, AI Care, 병원·약국 검색, 지도 연결 등 일부 기능을 제공하기 위해 Netlify Functions, OpenAI API, 지도/검색 관련 외부 서비스가 사용될 수 있습니다. 이 과정에서 사용자가 입력한 문장, 증상 정보, 검색 위치 정보 등이 기능 제공에 필요한 범위에서 처리될 수 있습니다.

9. 국외 이전 가능성

일부 외부 API 또는 클라우드 서비스는 해외 사업자 또는 해외 서버를 통해 제공될 수 있습니다. 사용자가 AI Care, 번역, 지도/검색 기능을 사용하는 경우 입력 정보가 해당 기능 제공을 위해 국외에서 처리될 가능성이 있습니다.
