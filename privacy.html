개인정보처리방침

SOS Bridge는 해외여행 중 갑작스러운 아픔이나 부상 상황에서 도움 요청을 돕는 참고 안내 서비스입니다. SOS Bridge는 의료기관, 응급기관 또는 공공기관이 아닙니다.

SOS Bridge는 의료진의 진단이나 치료를 대체하지 않습니다. 응급 상황에서는 즉시 현지 응급번호로 연락하거나 가까운 의료기관, 현지 공공기관, 숙소, 대사관 또는 영사관의 도움을 받으세요.

1. 서비스 소개

SOS Bridge는 한국어로 상황을 정리하고, 선택한 여행 국가/지역의 언어로 도움 요청 문장과 여행자 의료카드를 보여주는 보조 도구입니다. 사용자가 현지 의료진, 주변 사람, 숙소 직원, 공공기관에 상황을 설명할 수 있도록 돕는 것이 목적입니다.

2. 처리하는 정보

현재 앱 구조 기준으로 사용자가 직접 입력하거나 브라우저 기능을 통해 처리될 수 있는 정보는 다음과 같습니다.

  • 여행 국가/지역 선택 정보
  • 여행 도시/지역 입력 정보
  • 증상 입력 내용
  • 의료카드 입력 정보: 이름, 여권상 영문 이름, 국적, 나이, 혈액형, 알레르기, 복용 중인 약, 기존 질환, 비상 연락처, 여행자 보험, 숙소 주소
  • 사용자가 위치 권한을 허용하는 경우의 현재 위치 정보
  • 브라우저 또는 기기에서 자동으로 처리될 수 있는 기본 기술 정보

3. 정보를 사용하는 목적

  • 여행 국가/지역 설정과 현지어 도움 문장 표시
  • 여행자 의료카드 작성, 표시, 현지어 번역
  • 응급번호 정보와 정식 지원/제한 지원 상태 표시
  • 증상 입력을 바탕으로 한 AI Care 참고 안내 제공
  • 병원·약국 검색 기준 제공 및 지도 연결
  • 서비스 안정성 확인

4. 저장 여부

현재 베타 버전은 별도 회원가입 계정 서버나 데이터베이스에 의료카드 정보와 증상 정보를 저장하지 않습니다.

코드 확인 결과, 의료카드 정보와 의료카드 번역 캐시는 브라우저의 localStorage에 저장될 수 있습니다. 사용자는 앱 안의 “의료카드 정보 삭제하기” 버튼이나 브라우저 사이트 데이터 삭제 기능으로 이를 지울 수 있습니다.

코드 확인 결과, 현재 앱에서 sessionStorage 사용은 확인되지 않았습니다.

5. 외부 서비스 및 API 사용 가능성

현재 코드에서 확인되는 외부 처리 또는 연결은 다음과 같습니다.

  • Netlify Functions를 통한 AI Care 처리
  • Netlify Functions를 통한 의료카드 및 도움 문장 번역 처리
  • OpenAI API 처리: API 키는 브라우저에 노출하지 않고 Netlify 환경변수를 통해 사용됩니다.
  • 병원·약국 검색을 위한 Overpass API, Nominatim 검색, Google Maps 길찾기 링크
  • 사용자가 동의한 경우 브라우저 위치 권한

번역, AI Care, 병원·약국 검색 등 일부 기능을 사용할 때 입력 내용이 외부 API 또는 브라우저 기능을 통해 처리될 수 있습니다.

6. 민감정보 입력 주의

  • 알레르기, 복용 약, 기존 질환 등 건강 관련 정보는 꼭 필요한 범위에서만 입력하세요.
  • 공용기기에서는 사용 후 의료카드 정보를 삭제하거나 사이트 데이터를 지우는 것을 권장합니다.
  • 현지 의료진이나 주변 사람에게 화면을 보여줄 때 개인정보가 노출될 수 있습니다.
  • 비상 연락처를 입력할 때는 가능하면 해당 사람의 동의를 받으세요.

7. 제3자 제공

SOS Bridge는 사용자의 개인정보를 판매하지 않습니다. 별도 동의 없이 마케팅 목적으로 제공하지 않습니다.

다만 사용자가 번역, AI Care, 지도/검색 기능을 사용하는 경우 해당 기능 제공을 위해 입력 정보가 외부 서비스 처리 과정에 포함될 수 있습니다.

8. 이용자의 주의사항

  • 민감한 의료정보 입력 시 필요한 범위인지 먼저 확인하세요.
  • 공용기기에서는 입력 내용이 남지 않도록 주의하세요.
  • 응급상황에서는 앱만 의존하지 말고 현지 기관과 주변 사람에게 즉시 도움을 요청하세요.
